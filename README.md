# Tech Price Tracker

A Python project that tracks product prices using web scraping.

## Technologies

- Python
- Requests
- BeautifulSoup4

## Project Structure

```
tech_price_tracker/
│
├── data/
├── src/
├── tests/
├── README.md
└── .gitignore
```

## Current Features

- Read HTML files
- Parse HTML with BeautifulSoup
- Extract product name
- Extract product price

## Upcoming Features

- Scrape real websites
- Store data with SQLite
- Track price history
- Email notifications
